# Continuous Control Assurance – Essential Eight (E8) & SOCI

## Purpose
This repository implements a continuous control assurance program
aligned to the Australia's Essential Eight (E8) and Security of Critical Infrastructure (SOCI) Act using
GRC engineering principles.

## Design Principles
- Controls are decomposed into testable assertions
- Evidence is machine-generated
- Assurance is continuous, not point-in-time
- No screenshots or manual attestations

## Repository Structure
- docs/        Human-readable control intent and mappings
- data/        Machine-readable control definitions
- governance/  Assurance methodology and program governance
