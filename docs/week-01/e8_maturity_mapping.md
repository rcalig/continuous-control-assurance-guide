## Multi-Factor Authentication
### Base Assertions
MFA-01: All privileged users authenticate using MFA.
### Maturity Level 1
- MFA enforced for privileged users.

### Maturity Level 2
- MFA enforced for privileged and remote users.
- MFA enforcement cannot be bypassed via exclusions.

### Maturity Level 3
- MFA enforced for all users.
- MFA failures generate alerts and are monitored continuously.
